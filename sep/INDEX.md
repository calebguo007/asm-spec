# sep/ — Standard Extension Proposals

> ✅ This directory is pushed to the public repo.

SEP（Standard Extension Proposal）are ASM standard extension proposals for the MCP ecosystem, similar to Ethereum EIPs.

| File | Purpose | When to use |
|------|------|-----------|
| **sep-asm-service-value.md** | SEP Proposal: add Service Value annotations to MCP ToolAnnotations (price, quality, SLA) | When submitting to MCP official or citing in community discussions |

### SEP Numbering
- SEP-001: Service Value Annotations（current）
- SEP-002+: Reserved for future proposals (e.g., Trust Delta, Payment Channel)
